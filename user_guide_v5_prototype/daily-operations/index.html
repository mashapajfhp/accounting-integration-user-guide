🔧 Common Issues & Quick Solutions

❌ Sync failed with "Connection timeout" error
Quick Fix:
  1. Check your accounting system status page
  2. Wait 5 minutes and retry the sync
  3. If problem persists, contact support with error details
⚠️ "Split amount validation failed" in QuickBooks
Quick Fix:
  1. Check that all payroll items are mapped to accounts
  2. Ensure debit amounts equal credit amounts exactly
  3. Review cost center allocations for accuracy
  4. Re-run the sync after corrections
🔐 Authentication failed - need to reconnect
Quick Fix:
  1. Go to Settings → Payroll → Accounting Integration
  2. Click "Reconnect" or "Refresh Connection"
  3. Complete OAuth authentication flow
  4. Test connection before running sync
📊 Data appears incorrect in accounting system
Quick Fix:
  1. Check payroll item to account mappings
  2. Verify cost center assignments are correct
  3. Review exchange rates if using multiple currencies
  4. Contact support if discrepancies persist
🔄 Sync is taking much longer than usual
Quick Fix:
  1. Check if payroll size has increased significantly
  2. Verify accounting system performance
  3. Review API rate limits for your platform
  4. Consider syncing during off-peak hours
💡
Still need help? Visit our comprehensive troubleshooting guide or contact support with specific error messages.
